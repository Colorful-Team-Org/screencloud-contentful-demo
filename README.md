## Contentful Demo app

To run the app locally:

1. `npm install`

2. Update `src/config.development` with

```
{
    apiKey,
    mapName,
    playlistId,
    spaceId
}
```

3. `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
